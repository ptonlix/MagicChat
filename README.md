# MyGod

MyGod 是一个面向企业团队的 AI 原生协作产品，目标是把日常沟通、工作请求和 AI 助手放到同一个工作入口里。它不是单纯给聊天工具加一个机器人，而是希望让 AI 助手成为团队协作中的真实参与者，帮助用户处理消息、整理上下文、推进任务，并在需要人工判断时把工作交还给人。

产品当前聚焦企业内部协作场景：用户可以登录自己的工作空间，查看通讯录，与内置 AI 助手对话，并围绕聊天、联系人和任务管理展开工作。个人资料、头像和昵称等基础设置也会作为工作身份的一部分持续完善。

长期来看，MyGod 希望成为“助理优先”的企业工作入口。消息、邮件、会议、任务和业务系统中的请求可以先由个人助理理解、总结、分流和草拟处理；团队中的 Agent 也可以像成员一样参与讨论、承担任务、留下执行记录，并遵守清晰的权限和隐私边界。

这个项目还处在产品原型和能力验证阶段，重点是先打磨企业协作的核心体验，再逐步扩展到更完整的人机协作、任务流转、外部渠道接入和企业治理能力。
